# Polygon Portfolios

This is a demo nextjs app, used to interact with [the polygon api](https://polygon.io/)

## Requirements

- Node.js 16.8 or Later

## Getting Started

1. Add your Polygon API Key to `.env.local`
2. Install Dependencies
```bash
yarn install

# or

npm install
```
3. Run Locally with

```bash
yarn dev
#or
npm run dev
```
